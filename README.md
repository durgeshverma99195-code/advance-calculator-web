# 🧮 Advanced Responsive Calculator

This is my first **fully responsive calculator project** built using HTML, CSS, and JavaScript.
The calculator is designed to work smoothly across different screen sizes like mobile, tablet, and desktop.

## 🚀 Features

* Basic arithmetic operations (+, −, ×, ÷)
* Responsive design (works on all devices)
* Clean and modern UI
* Real-time calculation

## 📱 Responsiveness

This project is fully responsive and adapts to:

* Mobile devices 📱
* Tablets 📟
* Desktop screens 💻

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
